الهكر 
أفهم أنك مهتم بمجال الاختراق (الهكر)، لكن قبل البدء، لازم أوضّح لك شيء مهم جدًا:

⚠️ الاختراق مش بس "كسر حسابات" أو أشياء غير قانونية!
فيه "أخلاقيات الاختراق" (Ethical Hacking) وهو مجال أمن المعلومات لحماية الأنظمة، وده مشروع ومطلوب عالميًا.

لو قصدك الاختراق الضار (الهاكر الأسود)، فده جريمة يعاقب عليها القانون في كل الدول.

🛡️ لو قصدك تتعلم علشان تحمي (Ethical Hacking):
هبدأ معاك من الصفر بطريقة آمنة وقانونية:

1. تعلم الأساسيات أولًا:
كيف تشتغل الشبكات؟ (مثل: IP, DNS, Protocols).

تعلم نظام لينكس (مثل أوامر Terminal الأساسية).

فهم كيف تتعامل مع الأنظمة والبرمجيات (مثل: كيف يتم تخزين كلمات السر؟).

2. أدوات تحتاجها (قانونية):
Kali Linux: نظام تشغيل مخصص للاختبارات الأمنية.

Wireshark: لتحليل حركة الشبكة.

Metasploit: لأغراض اختبار الاختراق (بترخيص).

3. شهادات معتمدة:
CEH (Certified Ethical Hacker).

CompTIA Security+.

4. مواقع تتعلم منها (ببلاش):
TryHackMe (للمبتدئين).

Hack The Box (للمتقدمين).

❌ لو قصدك الاختراق الضار:
مفيش مساعدة هنا، لأنك ممكن:

تتعرض للمساءلة القانونية.

تضر ناس بريئة (ده ظلم وإثم).

تفقد مستقبلك الوظيفي في المجال التقني.

💡 بدائل ممتعة وقانونية:
تعلم كيف تحمي نفسك من الاختراق (مثل: كيف تصنع كلمة سر قوية؟).

تعلم برمجة وتطوير مواقع (تقدر تكسب منها فلوس!).

جرب تحليل البيانات أو الذكاء الاصطناعي.

🌈 الاختيار لك:

لو عاوز تتعلم الأمن السيبراني والاختراق الأخلاقي، ممكن أشرح لك بالتفصيل من البداية.

لو عندك فضول حول حماية حسابك الشخصي، ممكن أعلمك حيل أمان بسيطة.

أختر بإحترام.. وكن بطلاً في العالم الرقمي! 🚀

عاوز اعمل ارقام وهمية للواتساب ينفع تعملي رقم 
أنا آسف، لكن لا يمكنني مساعدتك في إنشاء أرقام وهمية أو تجاوز سياسات الخصوصية والتأمين الخاصة بالتطبيقات مثل واتساب.

لماذا؟ ⚠️
مخالف لشروط واتساب: استخدام أرقام وهمية قد يؤدي إلى حظر حسابك.

مشاكل أمنية: قد تُستخدم في أنشطة غير قانونية أو احتيال.

غير موثوق: معظم المواقع التي تقدم أرقام وهمية تكون احتيالية أو تنتهي صلاحيتها بسرعة.

✅ بدائل قانونية وآمنة:
شراء رقم مؤقت (حلال):

بعض المواقع تبيع أرقام حقيقية لاستخدام واحد (مثل مواقع SMS verification).

مثال: 5SIM أو Receive-SMS.

استخدام رقم ثانوي (مثل Google Voice):

إذا كنت في دول تدعمه (مثل أمريكا)، يمكنك إنشاء رقم مؤقت عبر Google Voice.

طلب مساعدة صديق:

يمكنك استخدام رقم شخص تثق به للتحقق المؤقت.

⚡ نصيحة مهمة:
إذا كنت تريد إخفاء هويتك لسبب مشروع (مثل الخصوصية)، يمكنك:

استخدام واتساب بزنس (مع بيانات حقيقية ولكن بغرض تجاري).

تفعيل إعدادات الخصوصية لإخفاء معلوماتك.

🚫 تجنب المواقع المشبوهة التي تعدك بأرقام وهمية مجاناً، لأنها قد:

تسرق بياناتك.

تخترق جهازك.

تُستخدم في عمليات غير قانونية دون علمك.

إذا كان لديك سبب مشروع لاستخدام رقم وهمي، اشرحه لي، وسأساعدك في إيجاد حل آمن وقانوني. 😊

انا عندي بيزنس اون لاين وطبعا العملاء بيثقه اكتر لما يتعامله علي الواتساب بالتالي عاوز رقم اربط بيه الواتساب وبالتالي هيكون الرقم ده بس انا ممكن اكون شرحتها بطريقة غلط بس لنا قولتلك رقم وهمي في نفس الوقت عاوزو مجاني علشان انا لسة ببدء 
فهمت قصدك الآن! 🚀 أنت تريد رقم واتساب خاص ببيزنسك لتبدو أكثر احترافية، لكنك تبحث عن حل مجاني لأنك في مرحلة البداية.

الحلول الآمنة والقانونية لك:
1. استخدم رقمك الشخصي + خاصية "واتساب بزنس" (مجاني تمامًا)
حمل تطبيق WhatsApp Business من جوجل بلاي أو آب ستور.

سجل بنفس رقم هاتفك الحالي (بدون حاجة لرقم جديد).

أضف معلومات بيزنسك (صورة، وصف، عنوان، إلخ).

الميزة: يظهر كحساب رسمي، ويمكنك فصل المحادثات الشخصية عن العمل.

2. اشترِ رقمًا حقيقيًا رخيصًا (ليس وهميًا)
في مصر: خطوط WE أو اورانج بـ10-20 جنيها شهريًا.

خارج مصر: مواقع مثل eSIM أو Hushed (أسعارها منخفضة).

3. استخدم رقم ثانوي مجاني (إن أمكن)
جرب تطبيقات مثل TextNow (يدعم أمريكا/كندا) أو 2ndLine.

ملحوظة: بعضها لا يدعم الواتساب، لكن يمكنك اختبارها.

4. استخدم "الرد الآلي" في واتساب بزنس
ضع رسائل ترحيب تشرح أن هذا الرقم للعمل فقط (يزيد الثقة).

⚠️ تجنب الأرقام الوهمية لأن:
قد تحظرها واتساب فجأة (خاصة إذا اكتشفت أنها وهمية).

العملاء قد لا يثقون برقم غير معروف أو مؤقت.

نصائح إضافية لبيزنس ناجح:
✅ أنشئ صفحة فيسبوك/إنستجرام واربطها بواتساب بزنس.
✅ استخدم رابط واتساب مباشر مثل:

Copy
https://wa.me/20123456789  
(استبدل بالأرقام الخاصة بك).

إذا واجهتك مشكلة في تفعيل واتساب بزنس، قل لي وأوجهك خطوة بخطوة! 😊

1. ما هي JavaScript؟
لغة برمجة تُستخدم لـ جعل صفحات الويب تفاعلية (مثل: ظهور القوائم، تحريك العناصر، إرسال النماذج دون إعادة تحميل الصفحة).

تعمل في المتصفح (مثل Chrome, Firefox) ويمكن استخدامها أيضًا في الخوادم (Node.js).

2. الأدوات المطلوبة:
متصفح (مثل Chrome).

محرر نصوص (مثل VS Code).

لوحة المطورين في المتصفح (تفتحها بـ F12 أو Ctrl+Shift+I).

Copy

1. حفظ الصفحة كـ PDF (أفضل حل سريع)

اختر "Save as PDF" بدل الطابعة.

على الموبايل:

إستخدم متصفح Chrome أو Edge واختر "مشاركة > طباعة > حفظ كـ PDF".

2. نسخ المحتوى إلى ملف Word أو Google Docs
إظهار كل الردود (بالتمرير لأسفل).

إختيار النص كله (Ctrl + A / Command + A).

نسخ (Ctrl + C / Command + C) ولصقه في ملف جديد.

3. أخذ لقطة شاشة (Screenshot) للجزء المهم
ويندوز: Win + Shift + S لاختيار جزء من الشاشة.

ماك: Command + Shift + 4 للسحب على المنطقة المراد حفظها.

4. حفظ الرابط في مكان آمن
إذا كان لديك رابط هذه الصفحة (URL)، يمكنك:

ما هو الدومين بيعمل ايه في الموقع 
ما هو الدومين (Domain)؟ وما وظيفته في الموقع؟
الدومين (اسم النطاق) هو عنوان موقعك على الإنترنت، مثل:

google.com

youtube.com

yourname.com

وهو بديل سهل عن عنوان الـ IP المعقد (مثل 192.158.1.38)، ليجعل الوصول لموقعك أسهل على المستخدمين.

🔵 ما الذي يفعله الدومين في الموقع؟
يُعرِّف الزائرين والمحركات (مثل جوجل) باسم موقعك

مثال: عندما يكتب أحدهم example.com في المتصفح، يتم توجيهه لموقعك.

يربط الاستضافة (الملفات المخزنة على السيرفر) بالعنوان الظاهر للجمهور

بدون دومين، سيكون الوصول لموقعك عبر IP صعبًا وغير احترافي.

يعزز الثقة والعلامة التجارية

الموقع بـ دومين مخصص (مثل shop.com) يبدو أكثر مصداقية من دومين مجاني (مثل my-site.wordpress.com).

🛠️ كيف يعمل الدومين تقنيًا؟
عندما يُكتب اسم الدومين في المتصفح:

يبحث المتصفح عن السيرفر المرتبط بهذا الدومين عبر نظام DNS (مثل دليل الهاتف).

يُوجه الزائر إلى الاستضافة (Hosting) حيث ملفات موقعك مخزنة.

يعرض الموقع للمستخدم.

javascript
Copy
// بيانات المنتجات (عادة تأتي من قاعدة بيانات)
const products = {
    1: { name: "عباية سوداء مطرزة", price: 499 },
    2: { name: "عباية كحلي مزينة", price: 599 }
};

// عرض محتويات السلة
function renderCart() {
    const cartItems = JSON.parse(localStorage.getItem('cart')) || [];
    const cartContainer = document.getElementById('cart-items');
    let total = 0;

    cartContainer.innerHTML = cartItems.map(item => `
        <div class="cart-item">
            <h3>${products[item.id].name}</h3>
            <p>السعر: ${products[item.id].price} جنيه</p>
            <div class="quantity-controls">
                <button onclick="updateQuantity(${item.id}, -1)">-</button>
                <span>${item.quantity}</span>
                <button onclick="updateQuantity(${item.id}, 1)">+</button>
            </div>
            <button onclick="removeItem(${item.id})">حذف</button>
        </div>
    `).join('');

    // حساب الإجمالي
    total = cartItems.reduce((sum, item) => sum + (products[item.id].price * item.quantity), 0);
    document.getElementById('total').textContent = total;
}

// تحديث الكمية
function updateQuantity(productId, change) {
    let cart = JSON.parse(localStorage.getItem('cart')) || [];
    const item = cart.find(item => item.id === productId);
    
    if (item) {
        item.quantity += change;
        if (item.quantity <= 0) cart = cart.filter(i => i.id !== productId);
        localStorage.setItem('cart', JSON.stringify(cart));
        renderCart();
    }
}

// حذف منتج
function removeItem(productId) {
    let cart = JSON.parse(localStorage.getItem('cart')) || [];
    cart = cart.filter(item => item.id !== productId);
    localStorage.setItem('cart', JSON.stringify(cart));
    renderCart();
}

// التحميل الأولي
window.onload = renderCart;

// عدّاد تنازلي للعرض
function updateCounter() {
    const endDate = new Date("2024-12-31");
    const now = new Date();
    const diff = endDate - now;
    
    const days = Math.floor(diff / (1000 * 60 * 60 * 24));
    document.getElementById("days").textContent = days;
}

// إضافة للسلة مع خاصية الكوليكشن
document.querySelectorAll(".add-to-cart").forEach(button => {
    button.addEventListener("click", function() {
        const productId = this.getAttribute("data-id");
        const cart = JSON.parse(localStorage.getItem("cart")) || [];
        
        cart.push({
            id: productId,
            collection: "new-2024",
            quantity: 1
        });
        
        localStorage.setItem("cart", JSON.stringify(cart));
        alert("تمت الإضافة إلى سلة التسوق!");
    });
});

// تشغيل عند تحميل الصفحة
window.onload = function() {
    updateCounter();
    setInterval(updateCounter, 86400000); // تحديث يومي

// نظام الإعجابات
document.querySelectorAll('.like-btn').forEach(btn => {
    btn.addEventListener('click', function() {
        const productId = this.getAttribute('data-product-id');
        let likes = JSON.parse(localStorage.getItem('likes')) || {};
        
        if (!likes[productId]) {
            likes[productId] = 1;
            this.classList.add('liked');
        } else {
            delete likes[productId];
            this.classList.remove('liked');
        }
        
        localStorage.setItem('likes', JSON.stringify(likes));
        this.querySelector('.like-count').textContent = likes[productId] || 0;
    });
});

// تأثيرات حركية عند التمرير
document.addEventListener('DOMContentLoaded', function() {
    // تأثير ظهور العناصر عند التمرير
    const animateOnScroll = function() {
        const elements = document.querySelectorAll('.team-member, .stat-card');
        
        elements.forEach(element => {
            const elementPosition = element.getBoundingClientRect().top;
            const screenPosition = window.innerHeight / 1.3;
            
            if (elementPosition < screenPosition) {
                element.style.opacity = '1';
                element.style.transform = 'translateY(0)';
            }
        });
    };
    
    // إعداد العناصر قبل الظهور
    const setupAnimation = function() {
        const elements = document.querySelectorAll('.team-member, .stat-card');
        
        elements.forEach(element => {
            element.style.opacity = '0';
            element.style.transform = 'translateY(20px)';
            element.style.transition = 'all 0.6s ease';
        });
    };
    
    setupAnimation();
    window.addEventListener('scroll', animateOnScroll);
    animateOnScroll(); // تشغيل مرة أولى عند التحميل
});

document.addEventListener('DOMContentLoaded', function() {
    // التحقق من صحة النموذج قبل الإرسال
    const contactForm = document.getElementById('contactForm');
    
    contactForm.addEventListener('submit', function(e) {
        e.preventDefault();
        
        // التحقق من صحة البيانات
        const name = document.getElementById('name').value.trim();
        const email = document.getElementById('email').value.trim();
        const subject = document.getElementById('subject').value;
        const message = document.getElementById('message').value.trim();
        
        if (!name || !email || !subject || !message) {
            alert('الرجاء ملء جميع الحقول المطلوبة');
            return;
        }
        
        // هنا يمكنك إضافة كود إرسال البيانات إلى الخادم
        // مثلاً باستخدام fetch أو axios
        
        // عرض رسالة نجاح
        alert('شكرًا لك! تم استلام رسالتك وسنتواصل معك قريبًا.');
        
        // إعادة تعيين النموذج
        contactForm.reset();
    });
    
    // تأثيرات حركية عند التمرير
    const animateOnScroll = function() {
        const elements = document.querySelectorAll('.contact-form, .contact-info, .map-section');
        
        elements.forEach(element => {
            const elementPosition = element.getBoundingClientRect().top;
            const screenPosition = window.innerHeight / 1.3;
            
            if (elementPosition < screenPosition) {
                element.style.opacity = '1';
                element.style.transform = 'translateY(0)';
            }
        });
    };
    
    // إعداد العناصر قبل الظهور
    const setupAnimation = function() {
        const elements = document.querySelectorAll('.contact-form, .contact-info, .map-section');
        
        elements.forEach(element => {
            element.style.opacity = '0';
            element.style.transform = 'translateY(20px)';
            element.style.transition = 'all 0.6s ease';
        });
    };
    
    setupAnimation();
    window.addEventListener('scroll', animateOnScroll);
    animateOnScroll(); // تشغيل مرة أولى عند التحميل
});

document.addEventListener('DOMContentLoaded', function() {
    // تفعيل/إلغاء تفعيل القوائم الفرعية
    const menuItems = document.querySelectorAll('.menu-item');
    
    menuItems.forEach(item => {
        const title = item.querySelector('.menu-title');
        
        title.addEventListener('click', function(e) {
            e.preventDefault();
            item.classList.toggle('active');
            
            // إغلاق باقي القوائم الفرعية
            if (item.classList.contains('active')) {
                menuItems.forEach(otherItem => {
                    if (otherItem !== item && otherItem.classList.contains('active')) {
                        otherItem.classList.remove('active');
                    }
                });
            }
        });
    });
    
    // تفعيل فلتر الألوان
    const colorFilters = document.querySelectorAll('.color-filter');
    
    colorFilters.forEach(filter => {
        filter.addEventListener('click', function() {
            colorFilters.forEach(f => f.classList.remove('active'));
            this.classList.add('active');
            // هنا يمكنك إضافة كود تصفية المنتجات حسب اللون
        });
    });
    
    // تفعيل فلتر المقاسات
    const sizeFilters = document.querySelectorAll('.size-filter');
    
    sizeFilters.forEach(filter => {
        filter.addEventListener('click', function() {
            sizeFilters.forEach(f => f.classList.remove('active'));
            this.classList.add('active');
            // هنا يمكنك إضافة كود تصفية المنتجات حسب المقاس
        });
    });
    
    // تفعيل خيارات الترتيب
    const sortSelect = document.getElementById('sort');
    
    sortSelect.addEventListener('change', function() {
        // هنا يمكنك إضافة كود ترتيب المنتجات حسب الخيار المحدد
        console.log('تم اختيار:', this.value);
    });
    
    // تأثيرات حركية عند التمرير
    const animateOnScroll = function() {
        const elements = document.querySelectorAll('.product-card');
        
        elements.forEach(element => {
            const elementPosition = element.getBoundingClientRect().top;
            const screenPosition = window.innerHeight / 1.3;
            
            if (elementPosition < screenPosition) {
                element.style.opacity = '1';
                element.style.transform = 'translateY(0)';
            }
        });
    };
    
    // إعداد العناصر قبل الظهور
    const setupAnimation = function() {
        const elements = document.querySelectorAll('.product-card');
        
        elements.forEach(element => {
            element.style.opacity = '0';
            element.style.transform = 'translateY(20px)';
            element.style.transition = 'all 0.6s ease';
        });
    };
    
    setupAnimation();
    window.addEventListener('scroll', animateOnScroll);
    animateOnScroll(); // تشغيل مرة أولى عند التحميل
});

document.addEventListener('DOMContentLoaded', function() {
    // تفعيل/إلغاء تفعيل القوائم الفرعية
    const menuItems = document.querySelectorAll('.menu-item');
    
    menuItems.forEach(item => {
        const title = item.querySelector('.menu-title');
        
        title.addEventListener('click', function(e) {
            e.preventDefault();
            item.classList.toggle('active');
            
            // إغلاق باقي القوائم الفرعية
            if (item.classList.contains('active')) {
                menuItems.forEach(otherItem => {
                    if (otherItem !== item && otherItem.classList.contains('active')) {
                        otherItem.classList.remove('active');
                    }
                });
            }
        });
    });
    
    // تفعيل فلتر الألوان
    const colorFilters = document.querySelectorAll('.color-filter');
    
    colorFilters.forEach(filter => {
        filter.addEventListener('click', function() {
            colorFilters.forEach(f => f.classList.remove('active'));
            this.classList.add('active');
            // هنا يمكنك إضافة كود تصفية المنتجات حسب اللون
        });
    });

        });

        });

document.addEventListener('DOMContentLoaded', function() {
    // محاكاة البحث عن طلب
    document.getElementById('track-btn').addEventListener('click', function() {
        const orderNumber = document.getElementById('order-number').value.trim();
        
        if (!orderNumber) {
            alert('الرجاء إدخال رقم الطلب');
            return;
        }
        
        // هنا سيتم جلب بيانات الطلب من السيرفر
        // هذه مجرد محاكاة للوظيفة
        simulateOrderTracking(orderNumber);
    });
    
    // محاكاة جلب بيانات الطلب
    function simulateOrderTracking(orderNumber) {
        // عرض رسالة تحميل
        const orderStatus = document.getElementById('order-status');
        orderStatus.innerHTML = `
            <div class="loading">
                <i class="fas fa-spinner fa-spin"></i>
                <p>جاري جلب بيانات الطلب ${orderNumber}...</p>
            </div>
        `;
        
        // محاكاة تأخير الشبكة
        setTimeout(() => {
            // في الواقع هنا ستأتي البيانات من API
            const orderData = {
                status: 'preparing',
                products: [
                    { name: 'عباية سوداء مطرزة', price: 499, quantity: 1, size: 'M', image: '../images/women/abaya1.jpg' }
                ],
                shipping: 50,
                date: '25 مايو 2024 - 10:15 ص'
            };
            
            renderOrderDetails(orderNumber, orderData);
        }, 1500);
    }
    
    // عرض تفاصيل الطلب
    function renderOrderDetails(orderNumber, data) {
        let statusSteps = '';
        let productsList = '';
        
        // تحديد حالة الطلب
        const statusConfig = {
            received: { icon: 'fa-check', text: 'تم استلام الطلب', active: false },
            preparing: { icon: 'fa-tshirt', text: 'قيد التجهيز', active: true },
            shipping: { icon: 'fa-truck', text: 'الشحن', active: false },
            delivered: { icon: 'fa-home', text: 'التسليم', active: false }
        };
        
        // إنشاء خطوات التتبع
        for (const [key, step] of Object.entries(statusConfig)) {
            const isCompleted = key === 'received' || (key === 'preparing' && data.status !== 'received');
            const isActive = key === data.status;
            
            statusSteps += `
                <div class="step ${isCompleted ? 'completed' : ''} ${isActive ? 'active' : ''}">
                    <div class="step-icon"><i class="fas ${step.icon}"></i></div>
                    <div class="step-text">
                        <h3>${step.text}</h3>
                        <p>${key === 'received' ? data.date : (isActive ? 'جاري العمل على طلبك' : '')}</p>
                    </div>
                </div>
            `;
        }
        
        // إنشاء قائمة المنتجات
        data.products.forEach(product => {
            productsList += `
                <div class="product-item">
                    <img src="${product.image}" alt="${product.name}">
                    <div class="product-info">
                        <h3>${product.name}</h3>
                        <p>الكمية: ${product.quantity}</p>
                        <p>المقاس: ${product.size}</p>
                        <p class="price">${product.price} ج.م</p>
                    </div>
                </div>
            `;
        });
        
        // عرض كل البيانات
        document.getElementById('order-status').innerHTML = `
            <div class="status-progress">
                ${statusSteps}
            </div>
            
            <div class="order-details">
                <h2>تفاصيل الطلب #${orderNumber}</h2>
                <div class="products-list">
                    ${productsList}
                </div>
                <div class="order-summary">
                    <h3>ملخص الطلب</h3>
                    <div class="summary-row">
                        <span>الإجمالي</span>
                        <span>${data.products.reduce((sum, p) => sum + (p.price * p.quantity), 0)} ج.م</span>
                    </div>
                    <div class="summary-row">
                        <span>الشحن</span>
                        <span>${data.shipping} ج.م</span>
                    </div>
                    <div class="summary-row total">
                        <span>المبلغ النهائي</span>
                        <span>${data.products.reduce((sum, p) => sum + (p.price * p.quantity), 0) + data.shipping} ج.م</span>
                    </div>
                </div>
            </div>
        `;
    }
    
    // (اختياري) جلب رقم الطلب من URL إذا كان موجوداً
    const urlParams = new URLSearchParams(window.location.search);
    const orderParam = urlParams.get('order');
    
    if (orderParam) {
        document.getElementById('order-number').value = orderParam;
        document.getElementById('track-btn').click();
    }
});

🔥 Firebase ببساطة: سيرفر جاهز في السحابة (بدون تعقيدات!)

Firebase هو منصة تطوير تابعة لـ Google تقدم لك خدمات جاهزة لبناء وتشغيل تطبيقات الويب والموبايل بدون الحاجة لبرمجة سيرفر معقد. إليك ما يهمك كمالك موقع "الفراعنة":

🛠 ماذا يفعل Firebase لموقعك؟
قاعدة بيانات فورية (Realtime Database)

تخزين بيانات الطلبات والعملاء (يتم تحديثها لحظيًا دون تحديث الصفحة).

مثال: عندما يتغير حالة الطلب من "قيد التجهيز" إلى "تم الشحن"، يرى العميل التحديث فورًا.

الاستضافة (Hosting)

رفع ملفات موقعك (HTML, CSS, JS) على سيرفر Google بضغطة زر.

المصادقة (Authentication)

نظام تسجيل دخول العملاء باستخدام الإيميل/الجوال/حسابات Google.

الوظائف السحابية (Cloud Functions)

تشغيل كود خلفي مثل إرسال إيميل تأكيد عند إنشاء طلب جديد.

💡 لماذا Firebase مناسب لموقعك؟
لا تحتاج لخادم (Server) خاص: كل شيء يعمل على سحابة Google.

مجاني للبداية: حتى 1GB تخزين و10GB نقل بيانات شهريًا.

سهل التكامل: بضعة أسطر من الكود تضيف فيها Firebase لموقعك.

2. ثانياً: تعديل ملف order-status.js
استبدل كل محتوى الملف بهذا الكود:

javascript
Copy
document.addEventListener('DOMContentLoaded', function() {
    const trackBtn = document.getElementById('track-btn');
    const orderInput = document.getElementById('order-number');
    const orderStatusDiv = document.getElementById('order-status');

    trackBtn.addEventListener('click', function() {
        const orderNumber = orderInput.value.trim();
        
        if (!orderNumber) {
            alert('❗ من فضلك أدخل رقم الطلب');
            return;
        }

        // عرض رسالة تحميل
        orderStatusDiv.innerHTML = `
            <div class="loading">
                <i class="fas fa-spinner fa-spin"></i>
                <p>جاري البحث عن الطلب ${orderNumber}...</p>
            </div>
        `;

        // الاتصال بـ Firebase
        const orderRef = firebase.database().ref('orders/' + orderNumber);
        
        orderRef.once('value')
            .then((snapshot) => {
                const orderData = snapshot.val();
                
                if (!orderData) {
                    orderStatusDiv.innerHTML = `
                        <div class="error">
                            <i class="fas fa-exclamation-triangle"></i>
                            <p>لا يوجد طلب بهذا الرقم!</p>
                        </div>
                    `;
                    return;
                }

                // عرض بيانات الطلب
                renderOrder(orderNumber, orderData);
            })
            .catch((error) => {
                orderStatusDiv.innerHTML = `
                    <div class="error">
                        <i class="fas fa-exclamation-triangle"></i>
                        <p>حدث خطأ: ${error.message}</p>
                    </div>
                `;
            });
    });

    function renderOrder(orderNumber, order) {
        // تحويل حالة الطلب إلى خطوات
        const statusSteps = {
            'جديد': { icon: 'fa-shopping-cart', done: true },
            'قيد التجهيز': { icon: 'fa-box-open', done: order.status !== 'جديد' },
            'تم الشحن': { icon: 'fa-truck', done: order.status === 'تم الشحن' },
            'تم التسليم': { icon: 'fa-check-circle', done: order.status === 'تم التسليم' }
        };

        let stepsHTML = '';
        for (const [status, info] of Object.entries(statusSteps)) {
            stepsHTML += `
                <div class="step ${info.done ? 'completed' : ''} ${order.status === status ? 'active' : ''}">
                    <div class="step-icon"><i class="fas ${info.icon}"></i></div>
                    <div class="step-text">
                        <h3>${status}</h3>
                        ${order.status === status ? `<p>${new Date().toLocaleDateString('ar-EG')}</p>` : ''}
                    </div>
                </div>
            `;
        }

        // عرض كل شيء
        orderStatusDiv.innerHTML = `
            <div class="status-progress">
                ${stepsHTML}
            </div>
            <div class="order-details">
                <h2>تفاصيل الطلب #${orderNumber}</h2>
                <div class="product-info">
